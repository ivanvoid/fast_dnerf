# Fast-Dynamic-NeRF

## Setup
```
conda env create -f environment.yml
```

## Data
[Dynamic NeRF dataset](https://drive.google.com/file/d/1uHVyApwqugXTFuIRRlE4abTW8_rrVeIK/view)


## Run
Train:
```
. train.sh
```

Inference:
```
. inference.sh
```
