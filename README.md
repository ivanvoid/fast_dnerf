# Fast D-NeRF
Dynamic NeRF but fast. (wip)

## Install

`git clone https://github.com/ivanvoid/fast_dnerf`

`pip install -r requirements.txt`

## Data

[Dynamic NeRF dataset](https://drive.google.com/file/d/1uHVyApwqugXTFuIRRlE4abTW8_rrVeIK/view)

## Run
`. run.sh`
